# カテゴリ・タグ体系

## カテゴリ（セクション種別）

セクションの「何であるか」を示す。1セクションにつき1カテゴリ。

| slug | 表示名 | 説明 | アイコン |
|---|---|---|---|
| hero | ヒーロー / ファーストビュー | ページ冒頭のメインビジュアルセクション | Layout |
| navigation | ナビゲーション / ヘッダー | サイトナビゲーション、ヘッダーバー | Menu |
| feature | フィーチャー / 機能紹介 | サービスや製品の特徴・機能を紹介 | Sparkles |
| pricing | 料金テーブル / プライシング | 料金プランの比較表示 | CreditCard |
| testimonial | テスティモニアル / お客様の声 | ユーザーレビュー、推薦コメント | MessageSquare |
| cta | CTA / コールトゥアクション | ユーザーに行動を促すセクション | MousePointerClick |
| footer | フッター | ページ最下部のナビゲーション・情報 | PanelBottom |
| faq | FAQ / アコーディオン | よくある質問と回答 | HelpCircle |
| team | チーム紹介 | メンバー紹介セクション | Users |
| stats | 統計 / 数字セクション | 実績数字やKPIの表示 | BarChart3 |
| logo-cloud | ロゴクラウド / パートナー | 取引先・パートナーロゴの並び | Building2 |
| blog | ブログ / 記事一覧 | ブログカードの一覧表示 | FileText |
| contact | コンタクト / お問い合わせ | 問い合わせフォームセクション | Mail |
| gallery | ギャラリー / ポートフォリオ | 画像やプロジェクトの一覧表示 | Image |

## タグ（デザイン特徴）

セクションの「どういう特徴か」を示す。1セクションに複数タグ可。

| slug | 表示名 | 説明 |
|---|---|---|
| gradient | グラデーション | グラデーションを効果的に使用 |
| glassmorphism | ガラスモーフィズム | 半透明・ブラー効果 |
| dark-mode | ダークモード | ダークテーマのデザイン |

| bold-typography | タイポグラフィ重視 | 大胆なテキスト表現 |

| 3d | 3D要素 | 3D的な表現を含む |
| retro | レトロ / ブルータリズム | レトロやブルータリスト系のスタイル |
| neomorphism | ニューモーフィズム | 柔らかい凹凸表現 |

## カテゴリ追加のガイドライン

- 10個以上のセクションが溜まりそうな種別のみカテゴリ化する
- それ以下の粒度はタグで対応する
- カテゴリ名は日本語（スラッシュ区切りで別名併記）

## タグ追加のガイドライン

- AIがセクション生成時に自動で付与する
- 既存タグへのマッピングを優先し、新規タグは慎重に追加
- 1セクションあたり2〜5個のタグを目安にする
