# フルコミットワークフロー

以下の手順で実行してください：

1. `git status` で現在の状態を確認
2. `git add -A` で全変更をステージング
3. `git diff --staged` で差分を確認
4. 以下の形式でコミットメッセージを生成：
   - 日本語
   - 50文字以内の要約
   - 詳細な説明
   - 変更されたコンポーネント一覧
5. `git commit -m "..."` でコミット
6. 現在のブランチを確認し、`git push` でプッシュ

## 注意

- mainブランチへの直接プッシュ時は確認を取る
- コンフリクトがあれば報告して停止
