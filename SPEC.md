# UI Lab — プロジェクト仕様書

## 概要

**UI Lab** は、最先端のモダンUIセクションを収集・生成・展示するWebギャラリーサイト。
企業サイトやSaaSで使われている洗練されたUIセクションをカテゴリ別に閲覧でき、
AIが週次でオリジナルUIコンポーネントを自動生成して追加する。

- URL: https://ui-lab.jp
- 言語: 日本語メイン
- 想定ユーザー: フロントエンド開発者、UIデザイナー
- 収益化: なし（個人プロジェクト）

---

## コンテンツ体系

### 1. UIセクション（メインコンテンツ）

AIが生成したオリジナルUIセクション。Next.js + Tailwind CSSのコードを提供。

- セクション単位（ヒーロー、料金テーブル、ナビゲーション等）
- 1つのセクションに対して複数バリエーション（ライト/ダーク、レイアウト違い等）
- ワンクリックでコードコピー可能
- ライブプレビュー表示

### 2. サイト事例（サブコンテンツ）

実在するかっこいいWebサイトの紹介。

- スクリーンショット + URL + 短い解説
- 国内・海外問わず、ジャンル不問
- 初期は手動キュレーション（将来的にAI自動収集を検討）

---

## ページ構成

### トップページ `/`

- ギャラリーレイアウトで UIセクションを一覧表示
- カテゴリフィルタ（クエリパラメータ `?category=slug`）
- タグフィルタ（クエリパラメータ `?tags=slug`）
- 新着順ソート
- ダークモード / ライトモード切替

### UIセクション詳細ページ `/sections/[slug]`

- ライブプレビュー（iframe）
- コードブロック（ワンクリックコピー）
- 関連セクション表示
- カテゴリ・タグ表示

### サイト事例一覧 `/sites`

- サイト事例のギャラリー表示

### Aboutページ `/about`

- サイトの趣旨説明
- 技術スタック

### お気に入り `/favorites`

- ローカルストレージベースのお気に入り機能

## AI自動更新パイプライン

### Phase 1（MVP）: AIによるUIセクション生成

- 週1回、GitHub Actions Cronで自動実行（+ 手動トリガー対応）
- Claude Code CLI (`@anthropic-ai/claude-code`) でセクションを自律生成
- `scripts/generate-sections-prompt.md` のガイドラインに従いデザイン統一
- Next.js + Tailwind CSSの実コードとして出力
- プレビュー画像を自動生成（Playwright、ダーク/ライト両テーマ）
- ファイルとしてコミット、Vercelに自動デプロイ
- 完全無人運用

### Phase 2（将来）: サイト事例の自動収集

- Web上のかっこいいサイトを自動発見
- スクリーンショット自動撮影
- AI による分類・解説生成
- ※ Phase 1の安定運用後に着手

- 日本語フォント: Noto Sans JP

## 技術スタック

- **フレームワーク**: Next.js latest（App Router、src ディレクトリなし）
- **スタイリング**: Tailwind CSS v4
- **UIコンポーネント**: shadcn/ui
- **言語**: TypeScript（strict mode）
- **データ管理**: ファイルベースCMS（content/ + fs読み込み）
- **ホスティング**: Vercel
- **AI**: Claude Code CLI（UIセクション自動生成）
- **スクリーンショット**: Playwright
- **CI/CD**: GitHub Actions
- **パッケージマネージャ**: npm
