# UI Lab — プロジェクト仕様書

## 概要

**UI Lab** は、最先端のモダンUIセクションを収集・生成・展示するWebギャラリーサイト。
企業サイトやSaaSで使われている洗練されたUIセクションをカテゴリ別に閲覧でき、
AIが週次でオリジナルUIコンポーネントを自動生成して追加する。

- URL: https://ui-lab.jp
- 言語: 日本語メイン
- 想定ユーザー: フロントエンド開発者、UIデザイナー
- 収益化: なし（個人プロジェクト）

---

## コンテンツ体系

### 1. UIセクション（メインコンテンツ）

AIが生成したオリジナルUIセクション。Next.js + Tailwind CSSのコードを提供。

- セクション単位（ヒーロー、料金テーブル、ナビゲーション等）
- 1つのセクションに対して複数バリエーション（ライト/ダーク、レイアウト違い等）
- ワンクリックでコードコピー可能
- ライブプレビュー表示

### 2. サイト事例（サブコンテンツ）

実在するかっこいいWebサイトの紹介。

- スクリーンショット + URL + 短い解説
- 国内・海外問わず、ジャンル不問
- 初期は手動キュレーション（将来的にAI自動収集を検討）

---

## ページ構成

### トップページ `/`

- ギャラリーレイアウトで UIセクション・サイト事例を一覧表示
- カテゴリフィルタ（セクション種別: ヒーロー、ナビゲーション等）
- タグフィルタ（特徴: アニメーション、ダークモード、グラデーション等）
- 新着順ソート
- ダークモード / ライトモード切替

### UIセクション詳細ページ `/sections/[slug]`

- ライブプレビュー（iframe or インラインレンダリング）
- コードブロック（ワンクリックコピー）
- バリエーション切替タブ
- 関連セクション表示
- カテゴリ・タグ表示

### サイト事例詳細ページ `/sites/[slug]`

- スクリーンショット画像
- サイトURL（外部リンク）
- 短い解説テキスト
- 関連するUIセクションへのリンク（あれば）

### カテゴリページ `/categories/[slug]`

- 特定カテゴリに属するセクション一覧

### タグページ `/tags/[slug]`

- 特定タグに属するセクション一覧

### Aboutページ `/about`

- サイトの趣旨説明
- AI自動生成の仕組みについての簡単な説明

## AI自動更新パイプライン

### Phase 1（MVP）: AIによるUIセクション生成

- 週1回、自動実行（GitHub Actions / Cron）
- 最新のUIトレンドをWeb検索で調査
- トレンドに基づいてオリジナルUIセクションを生成
- Next.js + Tailwind CSSの実コードとして出力
- プレビュー画像を自動生成（Playwright等でスクリーンショット）
- DBに自動登録、サイトに反映
- 完全無人運用

### Phase 2（将来）: サイト事例の自動収集

- Web上のかっこいいサイトを自動発見
- スクリーンショット自動撮影
- AI による分類・解説生成
- ※ Phase 1の安定運用後に着手

## 初期コンテンツ

ローンチ時に以下を用意する（すべてAI生成）:

- UIセクション: 30個以上（各カテゴリ最低3個）
- サイト事例: 10個以上

### 初期カテゴリ（UIセクション）

1. ヒーロー / ファーストビュー
2. ナビゲーション / ヘッダー
3. フィーチャー / 機能紹介
4. 料金テーブル / プライシング
5. テスティモニアル / お客様の声
6. CTA（コールトゥアクション）
7. フッター
8. FAQ / アコーディオン
9. チーム紹介
10. 統計 / 数字セクション

### 初期タグ

- アニメーション
- ダークモード
- グラデーション
- ミニマル
- グリッドレイアウト
- タイポグラフィ重視
- イラスト付き
- ガラスモーフィズム
- 3D要素

## デザイン要件

- ダークモード / ライトモード両対応（システム設定に追従 + 手動切替）
- レスポンシブ（モバイル、タブレット、デスクトップ）
- サイト自体が「モダンで洗練されたUI」のショーケースとなるクオリティ
- 日本語フォント: Noto Sans JP or Inter + Noto Sans JP

---

## 技術スタック

- **フレームワーク**: Next.js latest（App Router、src ディレクトリなし）
- **スタイリング**: Tailwind CSS v4
- **UIコンポーネント**: shadcn/ui
- **言語**: TypeScript（strict mode）
- **DB**: Supabase（PostgreSQL）
- **ホスティング**: Vercel
- **AI**: Claude API（UIセクション生成）
- **スクリーンショット**: Playwright
- **CI/CD**: GitHub Actions
- **パッケージマネージャ**: npm
